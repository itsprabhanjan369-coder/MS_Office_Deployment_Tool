1. Download "MS_Office_Installer.ps1", "setup.exe", and "Run.cmd".
2. Place all three downloaded files into the exact same folder.
3. Run the "Run.cmd" file to launch the program.
4. Select your desired version from the popup (Office 365, 2024, 2019, 2016, or 2013).
5. Check the boxes for the apps you want to install (unticked apps will be excluded).
6. Verify your chosen configuration on the right-hand status page.
7. Choose your installation method:
   - For Offline: Click "Download (Offline)", then click "Install (Offline)".
   - For Online: Click "Online Install" if you have a fast internet connection.
   - Note: You can click "Download setup.exe from GitHub" if you need the source file.
8. Wait without closing the windows until you see the "Deployment completed" popup.
